Technologies

Over here in JWS we not only use open source technologies we have developed our own tech and return back to community, there are the technologies that developed by as so far

  • pico: small footprint asynchronous module definition (AMD), our core library
  • pico-app: entity-component based web client framework, based on pico amd, underscore and backbone
  • pico-api: web application backend, based on pico amd and node.js
  • lean: css framework for web application, based on pico-amd
